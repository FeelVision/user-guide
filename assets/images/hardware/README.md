# Hardware Images

This directory contains images related to the hardware setup and components.

## Image Placeholders

Add the following images to this directory:

- `module-front.jpg` - Front view of the Luckfox Pico module
- `module-back.jpg` - Back view of the Luckfox Pico module
- `mounting-bracket.jpg` - The mounting bracket
- `attached-to-glasses.jpg` - Module attached to glasses
- `button-layout.jpg` - Button layout diagram
- `usb-connection.jpg` - USB cable connection
- `packaging.jpg` - Product packaging
- `components.jpg` - All components laid out

## Image Specifications

- Format: JPG or PNG
- Width: 1200px minimum
- Height: 800px minimum
- File size: Under 500KB each
- Background: Clean, neutral background
