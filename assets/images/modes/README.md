# Mode Images

This directory contains images demonstrating different modes and their use cases.

## Image Placeholders

Add the following images to this directory:

- `default-mode.jpg` - Default mode in action
- `ocr-mode.jpg` - OCR mode reading text
- `navigate-mode.jpg` - Navigate mode in use
- `face-mode.jpg` - Face recognition working
- `currency-mode.jpg` - Currency detection
- `edu-mode.jpg` - Educational mode
- `narrate-mode.jpg` - Narration mode

## Image Specifications

- Format: JPG or PNG
- Width: 1200px minimum
- Height: 800px minimum
- File size: Under 500KB each
- Show app interface or real-world usage
