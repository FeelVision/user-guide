# App Screenshots

This directory contains screenshots of the FeelVision Android app.

## Image Placeholders

Add the following images to this directory:

- `home-screen.jpg` - App home screen
- `mode-selection.jpg` - Mode selection screen
- `face-management.jpg` - Face management screen
- `settings-screen.jpg` - Settings screen
- `debug-mode.jpg` - Debug mode view
- `capture-result.jpg` - Capture result display
- `people-list.jpg` - Enrolled people list

## Image Specifications

- Format: JPG or PNG
- Width: 1080px minimum
- Height: 1920px minimum (portrait)
- File size: Under 300KB each
- High-quality app screenshots
