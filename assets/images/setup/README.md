# Setup Images

This directory contains images showing the setup and installation process.

## Image Placeholders

Add the following images to this directory:

- `app-installation.jpg` - App installation screen
- `permission-grant.jpg` - Permission request screen
- `model-download.jpg` - Model download progress
- `language-selection.jpg` - Language selection screen
- `settings-menu.jpg` - Settings menu
- `face-enrollment.jpg` - Face enrollment process
- `connection-status.jpg` - Connection status indicator

## Image Specifications

- Format: JPG or PNG
- Width: 1200px minimum
- Height: 800px minimum
- File size: Under 500KB each
- App screenshots or setup photos
